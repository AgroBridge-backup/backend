# Doc 2: Recommended Learning Resources

> **Executive Summary:** This document provides a curated list of high-quality learning resources for our core technologies (TypeScript, Node.js, Prisma) and architectural concepts (Clean Architecture). It includes a recommended learning path for new developers to accelerate their onboarding and a link to our internal knowledge base. This guide is a key tool for fostering a culture of continuous learning and engineering excellence.
>
> ---
> **Last Reviewed:** 2025-11-20
> **Document Owner:** Engineering Lead

A well-defined learning path empowers our team, reduces dependency on senior engineers for basic questions, and ensures that best practices are adopted consistently across the organization.

### 2. Recommended Learning Path for New Developers

1.  **TypeScript:** Start with the official handbook.
2.  **Node.js & Express:** Follow the MDN guide.
3.  **Prisma:** Complete the "Get Started" guide.
4.  **Architectural Concepts:** Read the articles on Clean Architecture and Dependency Injection.

### 3. Core Technologies & Concepts

*   **Node.js & Express:**
    *   [Official Node.js Docs](https://nodejs.org/en/docs/)
    *   [MDN - Express/Node.js Introduction](https://developer.mozilla.org/en-US/docs/Learn/Server-side/Express_Nodejs)
*   **TypeScript:**
    *   [TypeScript Handbook](https://www.typescriptlang.org/docs/handbook/intro.html) (The single best place to start).
*   **Prisma (ORM):**
    *   [Prisma Docs](https://www.prisma.io/docs) (World-class documentation).
*   **Architectural Concepts:**
    *   [Clean Architecture by Robert C. Martin](https://blog.cleancoder.com/uncle-bob/2012/08/13/the-clean-architecture.html)
    *   [Dependency Injection by Martin Fowler](https://martinfowler.com/articles/injection.html)
